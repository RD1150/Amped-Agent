# AmpedAgent UI/UX Improvement Spec - Key Findings

## Priority Implementation Order (from spec)
1. Add "Today's Focus" primary card
2. Simplify Coach section (expandable insights)
3. Apply spacing system (24–32px padding)
4. Improve typography (line height + hierarchy)
5. Add progression tracker

## 1. CORE UX SHIFT (MOST IMPORTANT)
- Transform dashboard from "data dashboard" into a "guided execution system"
- Replace multiple competing sections (Priority, Momentum, Blocker, Opportunity) with ONE dominant "TODAY'S FOCUS" card
- Today's Focus card: Title + Task + Subtext (outcome-based) + CTA button [Complete Now →]
- All other insights move into a secondary expandable section

## 2. INFORMATION ARCHITECTURE
### Coach Says Section - Two Layers:
- Layer 1 (Visible): Today's Focus (Primary), Progress Summary, One CTA
- Layer 2 (Expandable): Momentum, Key Blockers, Opportunities
- Add "View Insights →" link

## 3. VISUAL HIERARCHY
- Primary card: Increase size, stronger shadow, subtle background tint (light gray or very light orange), increase padding
- Secondary content: Lighter text color, reduce font size, reduce visual weight
- Increase vertical spacing across entire layout

## 4. USER PROGRESSION SYSTEM
- Add Progress Tracker: [✓ Attract] → [In Progress: Engage] → [Locked: Convert]
- Creates direction, encourages continued use, adds gamification

## 5. OUTCOME-BASED MESSAGING
- Replace task-based language with results-driven language
- Instead of: "Complete your Authority Profile"
- Use: "Complete your Authority Profile to increase leads by 30–50%"
- Apply across ALL UI messaging

## 6. RIGHT PANEL UPGRADE
- Upgrade from passive/informational to Performance Panel
- Add: Leads generated (weekly), Content posted, Profile strength score, Next unlock milestone

## 7. WEEKLY PLAN REDESIGN
- Replace checklist with Sequential System
- Step 1 (Active) → Step 2 (Locked 🔒 until Step 1 complete) → Step 3 (Locked 🔒)

## 8. COLOR SYSTEM SIMPLIFICATION
- Orange = primary action
- Green = completed
- Gray = inactive
- Red = critical issues only
- Remove purple unless assigned a clear meaning

## 9. TYPOGRAPHY SYSTEM
- Primary font: Inter (Alternative: Plus Jakarta Sans)
- Page Title: 24–28px (600–700 weight)
- Card Titles: 18–20px (600 weight)
- Section Labels: 12–13px (600 weight, slight letter spacing)
- Body Text: 14–15px (400–500 weight)
- Micro Text: 12–13px (400 weight)
- Line Height: Body 1.5–1.6, Headings 1.2–1.3
- Slight letter spacing for labels: 0.02em–0.05em
- Primary text: #111827, Secondary text: #6B7280

## 10. SPACING SYSTEM (CRITICAL)
- Use 8pt grid system (multiples of 8)
- Card padding: 24–32px
- Internal spacing: 16–20px
- Space between cards: 24–32px
- Vertical rhythm: [Title] (8px) [Subtitle] (16px) [Next section]

## 11. SIDEBAR IMPROVEMENTS
- Increase spacing between items: 16–20px
- Add horizontal padding: 16–20px
- Slightly reduce secondary item font size

## 12. BUTTON & CTA DESIGN
- Padding: Vertical 10–12px, Horizontal 16–20px
- Font size: 14–15px, Weight: 500–600
- Minimum 16px above buttons

## 13. RIGHT PANEL SPACING FIX
- Add 16–24px spacing above and below progress bar
- Reduce label font size to 13px

## 14. ADVANCED FEATURE (HIGH IMPACT)
- Add "DO IT FOR ME" button
- Allow users to: Generate content, Complete profile sections, Create posts instantly
- Goal: Shift from advice tool → execution tool

## FINAL OBJECTIVE
Transform UI from "Informational dashboard" → "Guided execution system that drives daily action and results"
