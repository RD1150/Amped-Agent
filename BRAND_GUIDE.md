# Authority Content - Brand Execution Guide

## Brand Intent
Authority Content is a confidence product for real estate professionals. It should feel quietly premium, editorial, and human — never flashy or trendy. The goal is for users to feel comfortable putting their name on the content.

## Color System

- **Slate Charcoal** (#1E293B) - Primary Headlines
- **Soft Charcoal** (#374151) - Body Text  
- **White** (#FFFFFF) - Primary Background
- **Muted Gold** (#C8A24D) - Accent Only
- **Deep Slate** (#0F172A) - CTA / Buttons

**CRITICAL**: Gold is an accent, not a fill color. Use it sparingly for emphasis, dividers, or hover states only.

## Typography System

Current typography is strong and should largely remain unchanged. Only minor refinements are required to enhance editorial authority.

**Recommended Structure:**
- Headlines (H1/H2): Serif font, authoritative weight
- Body/UI text: Sans-serif, neutral and highly readable

**Suggested Size & Weight Ratios:**
- H1: 42–48px, weight 600–700
- H2: 28–32px, weight 600
- H3 / Section headers: 20–22px, weight 600
- Body text: 15–16px, weight 400–500
- CTA buttons: 15–16px, weight 600

Serif usage should be limited to H1, H2, and major section headers only.

## Pre-Launch Visual Checklist

✓ Serif used only for headlines
✓ Body text remains light and readable
✓ Headline/body contrast is clear
✓ Gold used sparingly and intentionally
✓ CTA buttons feel confident and decisive
✓ Page feels calm, premium, and uncluttered
✓ No trendy-driven or experimental typography changes

If all boxes above are checked, the design is complete. Do not continue refining beyond this point.
